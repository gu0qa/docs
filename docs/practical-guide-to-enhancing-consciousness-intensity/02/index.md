举个例子：就如戒烟，往往完全修正过来需要一定的周期，因为整个过程中，很容易因为各种诱因下，如饭后，思考事情，焦虑，某习惯行为等，习惯性触发想抽烟的欲望，这个过程中需要一直保持观察，并及时调整自己，如通过零食等代替品进行过渡。
